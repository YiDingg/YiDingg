4. Transient Simulation

To obtain better voltage gain and phase margin performance, we use a cascode current structure as the tail current source. The circuit is shown below:

Before the transient simulation, we conduct a dc operation point simulation to ensure the circuit is properly biased.

Then, set the input DM voltage to be 1mV (1kHz, 0V offset) with 0V CM level, and run the transient simulation. Here are the results:

Noticing that the amplitude of vAv_AvA​ is about 3uV, which is very minimal so that vAv_AvA​ can be seen as an ac-ground.

Another interesting fact is that vAv_AvA​ has twice the frequency of vin,DMv_{in,DM}vin,DM​ (also vout,DMv_{out,DM}vout,DM​), as shown below:

This is because the circuit is ideally symmetric and the variation of vAv_{A}vA​ has the same frequency as ∣vin,DM∣|v_{in,DM}|∣vin,DM​∣, i.e., twice the frequency of vin,DMv_{in,DM}vin,DM​.

5. Open-Loop Gain A_DM

Now, we use the ac sweep to obtain the open-loop gain ADMA_{DM}ADM​ and its bode plot.

In most single-ended amplifier situations, to achieve this kind of simulation with higher accuracy, we are expected to set the dc point via negative feedback, and add the ac input signal to the circuit. In this case, how do we isolate the dc operation point and the ac input signal? An extremely large capacitor and induction play the role.

However, in the differential circuit below, we can directly add the ac input signal to the circuit without any isolation or negative feedback. Here are the results:

The simulation results give that:
ADM=44.7787 dB,GB=21.01 MHz,GBW=1.868 GHz,PM=3.83 ∘\begin{gather} A_{DM} = 44.7787 \ \mathrm{dB} ,\quad \mathrm{GB} = 21.01 \ \mathrm{MHz} ,\quad \mathrm{GBW} = 1.868 \ \mathrm{GHz} ,\quad \mathrm{PM} = 3.83^{\,\circ} \end{gather}ADM​=44.7787 dB,GB=21.01 MHz,GBW=1.868 GHz,PM=3.83∘​​

Recall that the theoretical calculation gives ADM=44.8689 dBA_{DM} = 44.8689 \ \mathrm{dB}ADM​=44.8689 dB, which is very close to the simulation result.

7. Simple Forward Compensation

To achieve better frequency response performance, we add a forward compensation capacitor CfC_{f}Cf​ to the circuit. This method significantly improves the phase margin, while sacrificing BW and GBW.

Setting CfC_fCf​ to be 0, 1pF, 10pF and 100pF, the simulation results are shown below:

At Cf=10 pFC_f = 10 \ \mathrm{pF}Cf​=10 pF, simulation gives that:

Cf=0:BW=21.01 MHz,GBW=1.868 GHz,PM=3.83 ∘Cf=10 pF:BW=2.976 MHz,GBW=331.85 MHz,PM=93.7 ∘\begin{align} C_f &= 0:&& \mathrm{BW} = 21.01 \ \mathrm{MHz},\quad \mathrm{GBW} = 1.868 \ \mathrm{GHz},\quad \mathrm{PM} = 3.83^{\,\circ} \\ C_f &= 10 \ \mathrm{pF}:&& \mathrm{BW} = 2.976 \ \mathrm{MHz},\quad \mathrm{GBW} = 331.85 \ \mathrm{MHz},\quad \mathrm{PM} = 93.7^{\,\circ} \end{align}Cf​Cf​​=0:=10 pF:​​BW=21.01 MHz,GBW=1.868 GHz,PM=3.83∘BW=2.976 MHz,GBW=331.85 MHz,PM=93.7∘​​
There is an improvement in phase margin, but the BW and GBW is significantly reduced.
